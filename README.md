# loan-risk-analysis

End-to-end loan risk analysis using Jupyter Notebook. Includes data cleaning, outlier detection, exploratory data analysis (EDA), statistical testing, and predictive modeling insights.

## Project Overview
This project analyzes loan applicant data to understand risk patterns and identify key financial indicators that influence loan approval and risk scoring. The workflow includes:
- Data preprocessing & missing value handling
- Outlier detection using IQR
- Correlation analysis and visualizations
- Statistical hypothesis testing
- RiskScore distribution & segmentation
- Model-driven analytical interpretation
